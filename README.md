# NLP_Project - Game Tag Summary and Recommendation 


### 프로젝트 목표
- 게임 리뷰 데이터를 기반으로 사용자 성향을 분석하고, 이를 반영하여 게임의 요약 평가와 새로운 태그를 자동 생성하는 Review-Based Game Summary Tag Recommendation 모델을 제안
- 기존 게임 태그가 개발사의 의도나 마케팅 전략에 따라 설정되어 실제 이용자 인식과 괴리가 발생하는 문제를 해결하기 위해, 본 연구는 사용자 - 리뷰 경험 기반 태그 재정의를 목표로 한다.
- 사용자의 리뷰 이력과 문체를 분석하여 ‘서사 중심’, ‘자유도 중심’, ‘안정성 중심’ 등의 성향 벡터를 도출하고, 이를 리뷰 텍스트와 함께 입력하여 리뷰의 의미를 성향에 따라 다르게 해석한다. 이를 통해 게임의 장점 · 단점 · 선호 유저 성향을 요약함과 동시에, 실제 이용자 인식에 부합하는 새로운 게임 태그를 제시한다.

- 개발사 중심의 고정적 분류 체계를 보완하고, 사용자 경험 중심의 동적 태그 생태계를 구축함으로써 개인화된 게임 평가와 추천 시스템의 가능성을 제시를 목적으로 한다.
